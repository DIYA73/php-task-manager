# PHP Task Manager

A simple task manager web application built with PHP and MySQL.

## Features

- User authentication (login / register)
- Create tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Pending / Completed)

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript
